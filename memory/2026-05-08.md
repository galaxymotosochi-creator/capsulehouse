## Session 2 — Hero Animations Deep Dive (позже 8 мая)

### Context
User сказала "это уже ближе" к 3 первым вариантам (hero-particles, hero-gradient, hero-3d). Запросила идеи анимаций кроме "падающих точек" для hero-экрана 888.ESTATE.

### Key user feedback
- **"Ну вот кстати это уже ближе!"** — реакция на 3 первых героя
- **"Давай поиграем с вариантами анимаций, сейчас там падают точки, что можно придумать еще?"** — просьба расширить идеи, не ограничиваясь particles
- **"1 и 2 я думаю что так"** — подтвердила два направления: Premium и Dramatic
- **"Меня сейчас интересует именно анимация первой страницы, фон, то что видит клиент когда открывает сайт"** — чёткий фокус на hero-секцию, фон
- **"Завтра гляну"** — отложила просмотр

### Decisions
- Разделили анимации на 2 настроения:
  - **Premium** (светлый/тёплый) — Canvas-based
  - **Dramatic** (тёмный #070707, золото #c9a84c) — Three.js-based
- Создали 4 варианта (2 премиум + 2 драматичных):

### Variant 1 — Premium "Золотые пылинки" (saved)
- Файл: `/root/.openclaw/workspace/musk/888-hero-premium.html` (→ скопирован в `site/demos/888-hero-premium.html`)
- Технология: Canvas
- Эффект: тёплый кремово-песочный градиент, плавно перетекающий между оттенками + редкие золотые пылинки
- URL: https://htmlpreview.github.io/?https://raw.githubusercontent.com/galaxymotosochi-creator/capsulehouse/main/demos/888-hero-premium.html

### Variant 2 — Premium "Барханы" (Dunes) (saved)
- Файл: `/root/.openclaw/workspace/musk/site/demos/888-hero-dunes.html`
- Технология: Canvas
- Эффект: плавные синусоидальные волны (как песчаные барханы) с мерцающими золотыми бликами на гребнях, тёплая песочная палитра
- URL: https://htmlpreview.github.io/?https://raw.githubusercontent.com/galaxymotosochi-creator/capsulehouse/main/demos/888-hero-dunes.html

### Variant 3 — Dramatic "Туман + Искры" (saved)
- Файл: `/root/.openclaw/workspace/musk/888-hero-dramatic.html` (→ `site/demos/888-hero-dramatic.html`)
- Технология: Three.js (CDN: https://cdn.jsdelivr.net/npm/three@0.170.0/build/three.module.js)
- Эффект: звёздное поле, 2 слоя тумана (fog), редкие золотые частицы кружатся в темноте
- URL: https://htmlpreview.github.io/?https://raw.githubusercontent.com/galaxymotosochi-creator/capsulehouse/main/demos/888-hero-dramatic.html

### Variant 4 — Dramatic "Архитектурный Wireframe" (saved)
- Файл: `/root/.openclaw/workspace/musk/site/demos/888-hero-arch.html`
- Технология: Three.js
- Эффект: 3D-сцена с wireframe зданием (башня с пирамидальной вершиной), мерцающие золотые частицы, звёздное поле, сетка-пол, туман
- URL: https://htmlpreview.github.io/?https://raw.githubusercontent.com/galaxymotosochi-creator/capsulehouse/main/demos/888-hero-arch.html

### Deployment
- Репозиторий: `galaxymotosochi-creator/capsulehouse` (ветка main)
- Демо-файлы в `demos/` — залиты через git push
- GitHub Pages не включён (нет токена/gh auth). Используем htmlpreview.github.io + raw.githubusercontent.com
- Временный сервер поднимался через `python3 -m http.server` на порту 8000 (успешно)

### Branding (единое)
- Логотип: "888.ESTATE" (цифры золотые: класс gold-text)
- Шрифты: `Cormorant Garamond` (300, 600, 700) + `Montserrat` (200, 300, 400) через Google Fonts

- Responsive: breakpoint 768px

### User's other ideas (not yet built)
- Живые линии — линии сами рисуются как чертёж будущего дома
- Тени/свет — имитация движения солнца, меняющего освещение
- Маятник — ритмичное покачивание минималистичных элементов
- Morphing blob — аморфные фигуры перетекают друг в друга

## Session 3 — Three.js Unreliable from Russia → Full Canvas 2D Rewrite

### Context
После того, как Юлия Валерьевна выбрала варианты 3 и 4 (Dramatic), я усилил их эффектами: UnrealBloomPass, золотое свечение, подсветка снизу, лучи света. Но пользователь сообщила: **"я вообще не вижу анимации сзади ни на одном сайте. Сзади просто черный фон"**.

### Root Cause
Three.js CDN (`cdn.jsdelivr.net`) заблокирован/нестабилен из России. ES-модули через import maps не грузятся → анимации не запускаются.

### Decision
**Полный переход на чистый Canvas 2D API** — НИКАКИХ внешних зависимостей (zero external imports). Canvas 2D работает в любом браузере без CDN, даже офлайн.

### Canvas 2D Variants Created (4 шт)

**1. Golden Fog** — `888-hero-fog-canvas.html`
- Техника: размытые золотые сферы (radial gradients), ~50 полупрозрачных кругов разного размера
- Эффекты: мерцающая золотая пыль (мелкие точки), лучи света сверху (градиентные полосы с вращением), подсветка снизу (тёплый градиент)
- Размер: ~7.4 KB
- URL: https://htmlpreview.github.io/?https://raw.githubusercontent.com/galaxymotosochi-creator/capsulehouse/main/demos/888-hero-fog-canvas.html

**2. Golden Vortex** — `888-hero-vortex-canvas.html`
- Техника: 1500 частиц, закрученных в спираль с трейлом (шлейф)
- Эффекты: пульсирующая золотая сердцевина, частицы оставляют следы с затуханием alpha, вращение + расширение
- Размер: ~6.4 KB
- URL: https://htmlpreview.github.io/?https://raw.githubusercontent.com/galaxymotosochi-creator/capsulehouse/main/demos/888-hero-vortex-canvas.html

**3. Architectural Frame (Wireframe Tower)** — `888-hero-arch-canvas.html`
- Техника: 3D-проекция wireframe здания на Canvas 2D (ручной vertex projection, no Three.js)
- Эффекты: башня с пирамидальной вершиной вращается в перспективе, светящиеся окна (маленькие золотые прямоугольники) мерцают с разными фазами, звёздное поле (100+ звёзд), сетка-пол
- Размер: ~8 KB
- URL: https://htmlpreview.github.io/?https://raw.githubusercontent.com/galaxymotosochi-creator/capsulehouse/main/demos/888-hero-arch-canvas.html

**4. City of Light** — `888-hero-city-canvas.html`
- Техника: ~25 зданий (разной высоты и ширины), отрисованных на Canvas 2D с псевдо-3D перспективой
- Эффекты: окна теплятся на всех этажах (случайные паттерны с пульсацией), панорамный облёт (медленное горизонтальное смещение), звёздное поле
- Размер: ~8.5 KB
- URL: https://htmlpreview.github.io/?https://raw.githubusercontent.com/galaxymotosochi-creator/capsulehouse/main/demos/888-hero-city-canvas.html

### Three.js Enhanced Variants (тоже созданы, но могут не работать в РФ)

Для полноты были также созданы 4 усиленных Three.js варианта с UnrealBloomPass:
- `888-hero-dramatic-glow.html` — туман + свечение + 6 лучей + подсветка
- `888-hero-dramatic-vortex.html` — 2000 частиц в спирали
- `888-hero-arch-glow.html` — wireframe башня с BloomPass
- `888-hero-arch-city.html` — городской силуэт с окнами

⚠️ Эти варианты могут показывать только чёрный фон из России из-за блокировки cdn.jsdelivr.net.

### What User Actually Saw
- Three.js варианты → чёрный фон (CDN не загрузился)
- Canvas 2D варианты → анимация работает (успешно отправлены пользователю как финальные 4)
